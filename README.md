| - `pages/` | static page sources (keeps public URLs via front matter `permalink`) |

| - `news/` | news index and categories |
| - `single-post/` | article pages |
| - `assets/` | CSS, JavaScript, and images |
| - `_layouts/`, `_includes/` | templates |
| - `scripts/` | import/cleanup utilities |
